# BLACK-BOX
This is a different repository from my Writeups repository even though it's also a Writeup repository, only I take the engagements here in a fully Black Box engagement type : no walkthroughs, no Guided Mode, no AI, just going blind. I'm authorized to do internet searches for patches/Vulnerabilities or information (for example when I don't know what's the meaning of a filename, I searched for .ps1, .vim, etc...) and for software I could use (GodPotato, pywhiskers...).
For example in my first ever Black Box engagement which was Optimum on HackTheBox, I found a certain patch on the systeminfo, and I had to search typing the name of the patch,
then website:microsoft.support.com to find the approximate date for the patch to find actual exploits that are unpatched on the system.

I authorize myself to use VeXX a specialized python scripter I built, in case metasploit framework fails or I don't want to use it and I need a complex python script, however, as of now (Redelegate rooted) I used it only once.
This is to test my limits as a pentesting operator and build a strong solo methodology, and also humble myself.
